README

All team members: Yuanhao Li, Bruce Liu, Hamza Tariq, Daksh Raichura, Sanskruti Pradeep Pawar

This project analyzes global wine ratings to understand how production attributes sensory characteristics and regional factors influence consumer perception of quality. Using a large scale dataset of wines and user ratings we cleaned and standardized key variables created consistent groupings for grape varieties elaboration styles and geographic classifications and applied rating volume thresholds to ensure reliable comparisons. We then conducted exploratory analysis and built dashboards to highlight patterns in winery performance grape popularity regional strength and year to year vintage consistency. The results provide actionable insights for producers retailers and importers including opportunities to refine assortment strategy strengthen brand positioning and target marketing based on measurable consumer preferences.
